generateBytesAsBase64

suspend fun generateBytesAsBase64(size: Int, flags: Base64Flags = Base64Flags.Default): Result<String>

Generates specified number of random bytes and returns them encoded as a Base64 string. The default encoding:

  • Does not wrap lines

  • Includes padding characters

  • Uses standard Base64 alphabet (not URL-safe)

Return

Result containing Base64 string. Length will be ceil(size * 4/3) with padding

Parameters

size

Number of random bytes to generate (must be positive)

flags

Base64 encoding configuration. Use Base64Flags constants

Throws

IllegalArgumentException

if size is not positive
